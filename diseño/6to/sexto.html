PASOS


CREAR UNA PAGINA WEB

utilice el programa de BLUEGRIFFON para poder crearlo. Abrimos uno nuevo y podemos darle cualquier color a nuestra pagina, solo es cuestion de escribir lo que queramos en ella ademas de que podemos crea tablas,poner listados e insertar imagenes.
cuando tenemos esto y esta diseñada  a tu gusto solo damos en archivo - guardar como y tu pagina se guarda en HTML


CREAR IMAGEN CORPORATIVA

la imagen corporativa es como lo ve el consumidor y como se nota lo que hacen.
si tienes una pequeña empresa tu imagen debe de ser simple para que la reconozcan, para crear tu logo puedes utilizar como referencia a lo que se dedica tu empresa. por ejemplo si es un estudio de fotografia poedes usar el lente de una camara y escribir el nombre de tu estudio o tambien puedes utilizar un monograma, el lema de tu imegen tambien es importante, debe de llamar la atencion de el publico.




En este semestre aprendimos a usar las herramientas de efectos donde pudimos crear paginas web, asi como tambien utilizamos el espacio de trabajo llamado BLUEGRIFFON donde se encontraban muchas herramienas para crear y diseñar tu pagina web.
